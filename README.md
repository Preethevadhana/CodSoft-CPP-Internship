
CodSoft C++ Internship

This repository contains the C++ internship tasks completed as part of the CodSoft Internship Program.

Tasks

Task 1: Number Guessing Game

A console-based game where the computer generates a random number between 1 and 100, and the user tries to guess it with hints provided after each attempt.

Concepts Used:

- Random Number Generation
- Loops
- Conditional Statements
- User Input Handling

Task 2: Simple Calculator

A calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

Concepts Used:

- Switch Statements
- Arithmetic Operators
- User Input/Output
- Conditional Statements

Task 3: Tic-Tac-Toe Game

A two-player console-based Tic-Tac-Toe game where players take turns marking spaces on a 3×3 board.

Concepts Used:

- Arrays
- Functions
- Loops
- Conditional Statements
